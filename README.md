## Why This Tool?

Wondering if you're paid fairly as a foreign engineer in Japan? Browse salary data by job role, city, and experience level.

## Features

- **Filter by role** — Software Engineer, Data Scientist, DevOps, and more
- **Filter by city** — Tokyo, Osaka, and other major tech hubs
- **Filter by experience** — Junior, mid-level, and senior ranges
- **Sortable data table** — Sort by any column to find what you need
- **Salary ranges** — See typical compensation bands, not just averages

## How It Works

1. Select your filters (role, city, experience)
2. Browse the salary data table
3. Compare ranges across different positions

> **Try it now:** [https://humancronadmin.github.io/japan-tech-salary/](https://humancronadmin.github.io/japan-tech-salary/)

## Privacy

- All data loads **in your browser** — no data is sent to any server
- No cookies, no account required
- Works offline after first visit (PWA)
